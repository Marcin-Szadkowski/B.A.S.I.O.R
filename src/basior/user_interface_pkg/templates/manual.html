Simulation is mainly based on libraries osmnx, networkx, geopandas, leaflet, folium and flask framework. Data such as tram routes and Wroclaw tramline graph structure are imported from OSM. Based on data mentioned above application even increase amount of data by self-developed mapper tool . Using tool we can define tram communication loops and check behaviour of a tramline graph when edges deletion occurs. You can run mapper_app.py in tools_pkg.

After starting application,process of loading data can take some time. Application is ready to use when trams appear on the map.

Choose symulation speed in dropbox positioned above map and click submit.

Tram line breaking simulation. To break tram's communication node, just choose apropriate coordinates by clicking on map and then submit.

To clear the map, type "None" or choose "None" from dropdown.

To visualize single tram route. Choose line from dropbox and click submit.
